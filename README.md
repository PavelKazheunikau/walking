Всем привет! Мой учебно-портфолийный проект для моей основной работы.
Данный скрипт считает площадь всех тифов в  данной папке и разпределяет на 3 категории - по количеству видоа печптных машин.
Моя основная профессия - печатник широкоформатной печати на рекламно-производственной компании.
Информация о заказах размазана по разным источникам -1с, Юджайл,мессенджеры. Собрать всю информацию их этих источников проблематично.
Поэтому мною было принято решение решить задачу в лоб. Просканировать все файлы за опредленный месяц, выбрать печатные файлы, посчитать площадь всей печати за месяц.
В зависмости от названия файла отнести его к определенной категории и вывести статистику по каждому виду печати. Н уи записать в файл (для начальства)
